──────────────────────────────────

──────────────────────────────

──────────────────────────────────────────

🔹 TASK B: Selenium Automated Form Testing
──────────────────────────────────────────

💻 STEP 1: Write Selenium Test Script (test.js)
───────────────────────────────────────────────
const {Builder, By} = require('selenium-webdriver');

(async function(){
let driver = await new Builder().forBrowser('chrome').build();

// Open local form
await driver.get('file:///C:/path/index.html');

// Locate and fill Name input
await driver.findElement(By.xpath('//input[@placeholder="Name"]')).sendKeys('Akhil');

// Locate and fill Email input
await driver.findElement(By.xpath('//input[@placeholder="Email"]')).sendKeys('test@gmail.com');

// Locate and click Submit button
await driver.findElement(By.tagName('button')).click();

await driver.quit();
})();

▶️ STEP 2: Execute Test Script
──────────────────────────────
node test.js

📊 STEP 3: Verify Test Results
───────────────────────────────
- Browser opens
- Form loads
- Name field filled: "Akhil"
- Email field filled: "test@gmail.com"
- Submit button clicked
- Success alert displayed ✅
- Browser closes automatically

🧠 KEY FLOW: open → locate → fill → click → verify → close
